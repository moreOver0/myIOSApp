# myIOSApp - Sliding Puzzle 滑动拼图

## 文件内容说明
1. SlidingPuzzle文件夹为整个工程
2. presentetion.m4v为app的演示视频

## App功能描述
Sliding puzzle 是种在平面上滑动数块平滑的板块以拼成特定排列的智力游戏，本应用中的3x3的滑动拼图又称重排九宫。  
本App志在提供一个简洁益智富有趣味性的游戏体验，仅提供如下两种功能
 
1. 撤销 (Undo)  
2. 新游戏 (New Game)  

**简单实用，消磨时间，居家旅行必备**  

## 其他的
